# Roadmap Tracker

A simple and interactive Roadmap Tracker built using HTML, CSS, and JavaScript. The application helps users track learning goals, monitor skill progress, and visualize completion status through a dynamic dashboard and progress bar.

## Features

### Learning Goal Management
- Add new skills
- Delete skills
- Mark skills as completed
- Undo completed skills

### Category Tracking
- Frontend
- Backend
- DSA
- DevOps
- AI/ML

### Progress Tracking
- Dynamic progress bar
- Completion percentage
- Track learning progress visually

### Dashboard Statistics
- Total Skills
- Completed Skills
- Remaining Skills

### Data Persistence
- Save roadmap data using Local Storage
- Automatically restore data after page refresh

### User Interface
- Clean and responsive design
- Beginner-friendly layout
- Easy skill management

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Roadmap-Tracker.git
```

2. Navigate to the project directory

```bash
cd Roadmap-Tracker
```

3. Open `index.html` in your browser

## How It Works

1. Enter a skill name.
2. Select a category.
3. Click **Add Skill**.
4. Mark skills as completed as you learn them.
5. Monitor progress using the dashboard and progress bar.
6. Delete skills that are no longer needed.
7. Data is automatically saved in Local Storage.
